                # 🌘学习路程

                我原来是一个计算机小白，当初连Linux都不懂的人，到后来慢慢的开始不断的学习，开始接触新的东西，一路走来学的东西总是杂乱无章，也有很多的知识忘记了，也有记忆深刻的知识。

                我的学习路程如下：

                • 接触过Linux操作系统,⽐如常⻅的RedHat、 CentOS、 Ubuntu 等操作系统后就开始学会玩服务器，自己捣鼓一些服务集群技术，后来接触了负载均衡、高可用、心跳检测等。
                • 学完艰苦的Linux课程之后，开始了我的云计算之旅，我在学习云计算的期间，学习了很多以往和现在比较主流的技术。
                  • 私有云的： OpenStack以及KVM虚拟化技术,包括常⻅的Vmware虚拟化技术
                  • 容器云的： Docker 、Kubernetes 主流的容器化操作
                  • 微服务： 学习过Istio的微服务治理以及CICD的Devops流程,Jenkins+Gitlab的技术,包括流⽔线的⼯作流程
                  • 容器虚拟化技术： KubeVirt未来趋势
                  • ⾃动化Ansible技术 分发集群部署服务
                  • Prometheus + Grafana 的监控集群技术有学习过
                  • 语⾔开发shell 和 python
                  • 包括常⻅的中间件服务Redis Rabbitmq Mysql 等
                • 我现在已经转开发，目前在从事的是区块链技术与应用开发和后端开发，在学习中。
